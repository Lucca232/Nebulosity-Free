# Nebulosity

## Overview

Nebulosity is a software application designed for image acquisition and processing, primarily aimed at Windows users working with astrophotography. The project facilitates capturing, calibrating, and stacking astronomical images, offering a structured workflow to assist in producing clear and well-processed results.

## Key Features

**Image acquisition from various camera models**  
**Support for multiple file formats including FITS and TIFF**  
**Calibration tools for dark frames, flat fields, and bias frames**  
**Image stacking with alignment and noise reduction**  
**Basic image processing functions such as stretching and cropping**  
**User interface tailored for clarity and ease of use**  
**Automation features to streamline repetitive tasks**

## Why Nebulosity?

Nebulosity provides a straightforward approach to astrophotography image processing, focusing on reliability and usability. The software is designed to offer clear workflows that help users manage complex image datasets efficiently. Its compatibility with a range of cameras and file formats ensures flexibility for various astrophotography setups, while maintaining stable performance on Windows platforms.

## Benefits

Using Nebulosity allows users to efficiently process astronomical images with tools that address common challenges in astrophotography. The software supports consistent calibration and stacking methods, which contribute to improved image quality and reduced noise. Its design emphasizes clarity in the user experience, enabling smoother operation without requiring extensive technical knowledge.

This repository is developed specifically for Windows systems, providing stable performance and efficient operation within this environment. The software leverages Windows capabilities to handle hardware integration and processing tasks effectively.
